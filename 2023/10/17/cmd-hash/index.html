命令行计算文件哈希值

2023/10/17

SHA2-256

Windows

powershell

Get-FileHash filename

Linux

sha256sum filename

macOS

shasum -a 256 filename

MD5

Windows

powershell

Get-FileHash -Algorithm MD5 filename

SHA-1

Windows

powershell

Get-FileHash -Algorithm SHA1 filename
